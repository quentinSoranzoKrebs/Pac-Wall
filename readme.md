# PacWall game

Pac-Wall est un jeu de plateforme inspiré de PacMan. Il peut être installé sur linux, windows et MacOS.

## Install hint

### In Linux

Clone the repos with git:
```bash
git clone https://gitlab.com/soranzokrebsquentin/pacwall.git
```

Install with cmake:
```bash
Cmake .
```
